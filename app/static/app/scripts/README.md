# Tibbiyot Halollik Tizimi

## Fayllar

Bu papkada 3 ta fayl mavjud:
1. `index.html` - Asosiy HTML fayl
2. `style.css` - CSS stil fayllari  
3. `script.js` - JavaScript kod

## Ishga tushirish

Oddiy brauzerda `index.html` faylni oching yoki HTTP server ishga tushiring:

```bash
python3 -m http.server 8080
```

Keyin brauzerda: `http://localhost:8080`
